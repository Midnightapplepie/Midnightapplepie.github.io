JavaScript

Expression: assigning a value to a variable, like: var me = "Kevin"

Function Expression: var thisVariable = function() {...}

Function Declaration: function thisVariable() {...}

Constructor: function ThisObject (parameters) { this.property=...; };
A constructor is to be used with the 'new' keyword to create instances.
Example:
function Car(brand,color){ this.brand=brand; this.color=color;};
var myCar = new Car ('Toyota','white');
myCar #=> { brand: 'Toyota', color: 'white' }

What happens here when we do 'new Car'?

  1. create new object myCar
  2. Car is a function (returns [Function: Car])
  3. myCar.constructor (returns [Function: Car]) == Car #=> true
  4. Car.prototype is an empty {}
  5. myCar is linked to Car.prototype
  6. myCar instanceof Car #=> true
Add properties and methods to instances:
Car.prototype.wheels = 4;
Car.prototype.drive=function() { return 'zoom...' };

myCar.wheels #=> 4
myCar.drive() #=> "zoom..."
now what:
  1. Car.prototype #=> { wheel: 4, drive: [Function] }
  2. myCar #=> { brand: 'Toyota', color: 'white' }
Nothing is Added to myCar but when you call myCar.wheels. it lookup wheels property and found nothing so it goes to Car.prototype.wheels and found 4. same with the function. Changes are found in object.constructor.prototype (myCar.constructor.prototype)
